## üìò About This Repository

This repository presents a curated collection of literature referenced in our comprehensive survey on image-based deepfake detection methods. These methods tackle the detection of synthetic faces generated by GANs, VAEs, diffusion models, and other generative architectures using deep learning and classical techniques.
